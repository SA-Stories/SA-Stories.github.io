San Andreas Stories - Territories Zones and Turf Map
====
[Live-map](https://skyrossm.github.io/np-gangmap/)
Este mapa interativo (Google) mostra a localização das atuais zonas de território/gangue.

## Como enviar uma nova localização?
1. Clique com o botão direito do mouse no mapa para adicionar pontos.
2. Clique em "Adicionar nova região" para adicionar os dados da região.
3. Substitua "\<edit this\>" por informações corretas.
4. Substitua o ID por um ID não utilizado enquanto tenta manter o padrão em `locations.json`
5. Altere o tipo, adicione um vídeo ou imagens e altere a cor da região.
6. Adicione nova região ao `locations.json` e crie uma solicitação de pull.

Alternatively a new issue can be created the region data and it will be added.

## How to host yourself

1. Clone this repository
2. Host the repository using Nginx, Apache, or another webserver.

To host the repository using Python 3 use in the root of the repository:
```
python -m http.server
```
